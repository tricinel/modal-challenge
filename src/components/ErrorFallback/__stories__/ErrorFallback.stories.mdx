# ErrorFallback

The `<ErrorFallback />` is a special component that handles runtime errors in your application. You can read more about it [here](https://kentcdodds.com/blog/use-react-error-boundary-to-handle-errors-in-react).

> **Error boundaries do not catch errors for:**
>
> * Event handlers
> * Async code
> * Server side rendering
> * Errors thrown in the Fallback component itself (which is why we keep that one simple)
